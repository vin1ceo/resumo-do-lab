# resumo-do-lab

## Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO

No bootcamp foi aprendido que dentro da plataforma da AZURE, possuí diversos serviços para o ambiente coorporativo, para realocar aplicações em nuvens.
- Foi ministrado os tipos de nuvens.
- IaaS: Infraestrutura total em nuvem, ao invés de comprar todo hardware em um ambiente local, é possível ter este recurso de forma virtualizada, onde é possível ter total controle do hardware.
- SaaS: (Software como Serviço) ao contrário do IaaS ele é algo já desenvolvido, O usuário não precisa se preocupar com a infraestrutura, segurança ou atualizações; o provedor cuida de tudo. Ideal para: Usuários finais que precisam de uma solução de software específica sem o incômodo da instalação ou manutenção, economizando tempo e dinheiro. 
Exemplo: Serviços como Gmail, Microsoft 365, ou sistemas de gestão empresarial (ERP) baseados em nuvem
- PaaS:  o PaaS (Platform as a Service) oferece uma plataforma para os desenvolvedores criarem seus próprios aplicativos, sem se preocupar com infraestrutura.


Nuvem pública: Serviços de nuvem compartilhados com o público em geral ou múltiplas organizações pela internet, como Google Drive e OneDrive.
Nuvem privada: Um ambiente de nuvem dedicado a uma única organização, oferecendo mais controle e segurança.
Nuvem híbrida: Combina nuvens públicas e privadas, permitindo o compartilhamento de dados e aplicações entre elas para otimizar desempenho e custos.
Multinuvem: Utiliza dois ou mais serviços de nuvem pública ou privada de diferentes provedores, permitindo otimizar desempenho e minimizar custos.
